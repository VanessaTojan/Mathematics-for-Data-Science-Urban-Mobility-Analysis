# Mathematics for Data Science Using Python

![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

## Project Overview

This project demonstrates the practical application of core mathematical concepts used in Data Science through a smart urban mobility case study.

The analysis combines Linear Algebra, Calculus, Probability, and Statistics to investigate traffic flow behaviour, vehicle acceleration patterns, autonomous vehicle reaction times, and speed compliance within intelligent transportation systems.

The project was implemented using Python and Jupyter Notebook, with mathematical modelling supported by numerical computation and data visualization.

---

## Objectives

- Model traffic flow equilibrium using systems of linear equations
- Analyze vehicle velocity and acceleration using calculus
- Evaluate uncertainty in autonomous vehicle reaction times using probability distributions
- Perform statistical analysis and hypothesis testing on vehicle speed data
- Demonstrate how mathematical techniques support data-driven decision making

## Project Results

### Traffic Flow Analysis
Solved a system of simultaneous equations to determine equilibrium traffic flows across four road networks.

### Vehicle Dynamics
Derived acceleration functions from velocity equations and identified maximum and minimum acceleration points.

### Autonomous Vehicle Reaction Times
Modelled reaction times using a normal distribution and calculated probabilities for rapid and delayed responses.

### Speed Compliance Analysis
Performed statistical testing to determine whether observed vehicle speeds exceeded legal speed limits.
